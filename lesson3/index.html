Site Purpose

The purpose of this weather website is to report the weather for Preston, Soda Springs and Fish Haven. Making this a simple website where residents and visitors can be warned of storms as well as sunny days. The home page will include rotating weather-related stories that will help the users have a better idea of how to use the type of weather they are experiencing. Each town will have its own page which will give individual weather information as well as information about each town to help the users get ideas of what to do including events, local hotspots and eateries. This will be a responsive website that the users can use on any computer, tablet or phone easily.
